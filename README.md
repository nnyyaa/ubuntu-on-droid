# ubuntu-on-dorid
a sc to install ubuntu on phone

这个脚本会自动在你的手机上安装ubuntu  
###### 注意，现只支持Ubuntu22.04

___

流程：

安装termux
```
pkg upd  
pkg upg  
pkg i git -y  
git clone https://github.com/wspbh/ubuntu-on-phone.git  
cd ubuntu-on-phone  
chmod +x *  
./ub-i.sh  
```
等等等


安装完成后输入startubuntu启动
